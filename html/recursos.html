📚 Recursos y curación de contenidos

En este apartado se recogen los recursos digitales que ayudarán al alumnado a desarrollar el proyecto y las herramientas que utiliza el profesorado para organizar la práctica docente.

🎧 Curación de contenidos para el alumnado

El alumnado consultará una recopilación de versiones, materiales de análisis y recursos de apoyo para preparar su interpretación.

  • Versiones de referencia de dúos para flauta y piano.
  • Indicaciones para analizar tempo, fraseo, balance y articulación.
  • Recursos para grabación y autoevaluación.

Enlace al Padlet del alumnado: https://padlet.com/cmforcada/analisis-de-versiones-7i55hbjpio0tr63r

🧰 Curación de contenidos para el docente

El profesorado dispone de una selección de herramientas digitales para gestionar el aula, organizar materiales y enriquecer el proceso de enseñanza-aprendizaje.

  • Google Classroom o plataforma equivalente.
  • Google Drive para organización documental.
  • IMSLP para consulta de partituras.
  • Metrónomo y afinador digital.
  • Herramientas de grabación y edición básica.

Enlace al Symbaloo docente: https://www.symbaloo.com/shared/AAAABhdEajAAA42ADtAj0g==

🧭 Uso responsable de los recursos

Los recursos se utilizarán de forma crítica, respetando la autoría, las licencias y la privacidad de las personas participantes.
